Grammar: Phrasal Verbs (Try on)

Many English verbs have two parts (Try + On). When we use "It", it MUST go in the middle!

I want to try on the shirt.

I want to try the shirt on.

I want to try it on. (Correct!)

I want to try on it. (Wrong!)

Teacher: This is a classic A2 rule. Pronouns like 'it' or 'them' MUST separate the verb.
